# Zak Portfolio (Vite + React)

A personal portfolio website built with **Vite + React** and styled with **Tailwind CSS**.  
This repo is meant to be a solid starting point — clone it, replace the content with your own info, and deploy it.

🌐 Live portfolio: https://berrizak.com

---

## Tech Stack

- **Vite** (React)
- **React Router**
- **Tailwind CSS**
- **Radix UI**
- **Framer Motion**
- **Lucide Icons / React Icons**

> Note: The project also includes some Hostinger Horizons/visual-editor related Vite plugins (used during development). :contentReference[oaicite:1]{index=1}
