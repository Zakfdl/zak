# Zak Portfolio (Vite + React)

A personal portfolio website built with **Vite + React** and styled with **Tailwind CSS**.

This project started as an AI-generated theme to speed up development. After generating the base structure, the source code was exported, customized locally, pushed to GitHub, and deployed.

The goal was to keep full control of the source code and update the portfolio anytime without being limited by platform editing restrictions.

Live Portfolio  
https://berrizak.com


## Tech Stack

This project uses modern frontend technologies:

- Vite
- React
- React Router
- Tailwind CSS
- Radix UI
- Framer Motion
- Lucide Icons
- React Icons


## How This Portfolio Was Built

The workflow was simple and practical:

1. Generate a portfolio layout using an AI tool.
2. Export the source code.
3. Customize and clean the code locally.
4. Push the project to GitHub.
5. Deploy the project to hosting.
6. Maintain full control of the source code for unlimited updates.


## Getting Started

Follow these steps to run the project locally.


### 1. Clone the repository

```bash
git clone https://github.com/Zakfdl/zak.git
cd zak
```


### 2. Install dependencies

```bash
npm install
```


### 3. Run the development server

```bash
npm run dev
```

Now open your browser and go to:

http://localhost:3000


## Build the Project

To create a production build:

```bash
npm run build
```

This command generates a **dist** folder that contains optimized production files.


### Preview the production build locally

```bash
npm run preview
```


## Deployment

After building the project, you can deploy the **dist folder** to any static hosting provider.

Typical deployment steps:

1. Run the build command

```bash
npm run build
```

2. Locate the generated folder

dist/

3. Upload the contents of the **dist** folder to your hosting provider.


### Supported Hosting Platforms

You can deploy this project to:

- Hostinger
- Netlify
- Vercel
- GitHub Pages
- Any static hosting service


## Customize the Portfolio

If you want to use this project as your own portfolio:

1. Update personal information (name, role, summary).
2. Replace the projects section with your own work.
3. Update experience and achievements.
4. Update social links (LinkedIn, GitHub, email).
5. Replace images and assets.

Most edits will be inside the **src** folder.
